# نصب هیدیفای روی داکر
توجه کنید نسخه داکر هنوز آزمایشی هست و ممکن است اشکالاتی داشته باشد که لطفا به ما اطلاع دهید یا لطفا پول ریکوئست  بزنید و اصلاح کنید
.نسخه داکر برای استفاده پیشنهاد نمیشود و [لطفا از نصب معمول استفاده کنید](https://github.com/hiddify/hiddify-config/wiki/%D9%86%D8%B5%D8%A8-%D8%B3%D8%B1%DB%8C%D8%B9-%D8%AF%D8%B1-%D8%A7%D9%88%D8%A8%D9%88%D9%86%D8%AA%D9%88)

برای داکر کافی است کد زیر را اجرا کنید:
</div>

```
git clone https://github.com/hiddify/hiddify-config
cd hiddify-config
docker-compose up -d
```
